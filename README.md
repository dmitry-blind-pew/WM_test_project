# Отчет о потреблении кофе

Скрипт читает один или несколько CSV-файлов с данными о подготовке студентов и выводит в консоль таблицу с медианной суммой трат на кофе по каждому студенту.  
Поддерживаемый отчет: `median-coffee`.

## Пример запуска

```bash
python main.py --files data/math.csv data/physics.csv data/programming.csv --report median-coffee
```
